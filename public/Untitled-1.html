Modelado de cajas o Model Box.

Es un modelo implementado en el lenguaje de programación de “CSS”; para determinar el modelo de tamaño, posición, propiedades de (color, fuente, tamaños, bordes); de las cajas a editar. Los elementos son representados en cajas rectangulares, para ser parte del diseño de los entornos (HTML) y ajustarse a la interfaz gráfica de las páginas web y cada rectángulo, almacena información, para la construcción del entorno.

Cada elemento diseñado en un entorno (HTML), se mostrará; cómo una caja rectangular y serán designados en un espacio de trabajo, según el requerimiento del programador.

Generalidades de Model box

Área de contenido: Conocida cómo área de relleno o padding área; delimita el área para extender o delimitar el contenido a incluir en un elemento. Se delimita con las dimensiones del ancho: (padding-box width) y alto: (padding-box height).

Área de borde: Delimita el relleno de un borde e incluye los elementos que lo delimitan.

Área de margen: Delimita y extiende el área de los bordes ; para incluir un área vacía y separándolos de los demás elementos.
